# Week3 App – DevOps Capstone Project

## Overview
End-to-end DevOps pipeline using Git, Jenkins, Docker, Helm, ArgoCD, and Kubernetes.

## CI/CD Flow
Git push → Jenkins CI → Docker build & push → Helm update → ArgoCD sync → Kubernetes deploy

## Tools Used
- Jenkins
- Docker
- Kubernetes
- Helm
- ArgoCD
- Git

## How to Deploy
1. Push code to Git
2. Jenkins builds and pushes Docker image
3. ArgoCD syncs application
4. App runs in Kubernetes
